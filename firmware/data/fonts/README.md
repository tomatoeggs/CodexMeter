# Fonts

`DroidSansFallback.ttf` is copied from AOSP:

https://raw.githubusercontent.com/aosp-mirror/platform_frameworks_base/master/data/fonts/DroidSansFallback.ttf

The Droid fonts are licensed under the Apache License 2.0. The font is used by
the LVGL TinyTTF prototype so the device can render Chinese text locally instead
of receiving pre-rendered images over BLE.
